# homebrew-tap

Official Homebrew tap for [KeyHarbour](https://keyharbour.ca).

## Install

```bash
brew install KeyHarbour/tap/kh
```

## Usage

```bash
kh login --token <your-api-token> --endpoint https://app.keyharbour.ca/api/v2
kh whoami
```
